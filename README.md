# MarbleRun
An analog-to-digital marble run converter.

## Colors
Blue = Spawn
Green = Trampoline
Red = Booster
Purple = Portal

## Arduino
Arduino is optional, but if you want to use one, use the following pins, or edit the code

Pin 8 - Button
Pin 9 - Button 
Pin A0 - Potentiometer 
Pin A1 - Potentiometer
